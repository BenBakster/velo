# velo v0.3 — Таблица заимствований (Borrowing Table)

Этот документ описывает внешние проекты, их лицензии, статус заимствования и интеграцию в проект **velo** на момент релиза v0.3.

## Правила заимствования (Licensing Policies)
- **Отсутствие лицензии (no LICENSE):** Только для ознакомления (read-only), копирование кода запрещено.
- **SecBSD / FuguIta / Лицензии без разрешения:** SecBSD выходит за рамки проекта (out of scope).
- **GPL лицензии:** Запрещены к включению в `base`/`site`.
